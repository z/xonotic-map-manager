.. Xonotic Map Manager Intro

Intro
=====

A command-line package manager for the xonotic-map-repository project.

Use it like ``apt-get``::

    xmm update
    xmm install dance2.pk3
    xmm list

Used by default with the unofficial Xonotic map repository `xonotic.co`_.
